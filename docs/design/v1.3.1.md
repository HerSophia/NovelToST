# v1.3.1 设计基线（可执行）

### 1) 新手友好 UI（设定工坊分区 + 帮助入口）

把当前“功能堆叠”调整为「步骤区块」：

- 步骤1：创建/管理会话  
- 步骤2：对话协作（扩写/精修/一致性/候选）  
- 步骤3：草案与锁定字段  
- 步骤4：世界书绑定与提交（含 Outline 同步）

每个关键区块右上加 `❓`（`HelpTriggerButton`），统一触发 Workbench 的 `HelpModal`。

---

### 2) 世界书绑定升级为基础能力（chat scope）

新增基础服务（不绑死在“提交目标”逻辑里）：

- `core/worldbook-binding.service.ts`（新）
  - `getChatWorldbookBindingSnapshot()`
  - `bindChatWorldbook(name)`
- 使用宿主 API：
  - `getChatWorldbookName('current')`
  - `rebindChatWorldbook('current', worldbook_name)`

并在 UI 增加“当前聊天已启用世界书”区块，显示状态 + 一键应用绑定。  
**提交目标默认跟随绑定结果**（自动写入 `worldbuildingStore.selectedWorldbookName`）。

---

### 3) 锁定字段中文展示（仅展示层）

在 `WorldbuildingPanel.vue` 内增加映射，不改底层 key：

- `name → 名称`
- `aliases → 别名`
- `summary → 摘要`
- `facts → 事实要点`
- `constraints → 约束`
- `relations → 关系`
- `extra → 扩展字段`

---

### 4) 文案“人话化”（中性清晰）

重点替换位置：

- `WorldbuildingPanel.vue`：按钮文案、空态说明、提交/同步提示
- `useWorldbuildingControl.ts`：toast 警告/错误信息

原则：短句、动作导向、给下一步建议；不动安全行为（解析失败保护、锁字段合并保护）。
